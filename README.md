# Repeater
Genome-wide tool for sensitive de novo identification of interspersed and tandem repeats

To run the project from the command line, go to the dist folder and type the following:

java -jar Repeater2.jar in.file kmer=12 min=50 quick=false flangs=true

for example:

java -jar Repeater2.jar d:\1.txt 

java -jar D:\Repeater2\dist\Repeater2.jar d:\1.txt kmer=21 min=100
